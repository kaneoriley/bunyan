# Gradle Dependency

Firstly, you need to add JitPack.io to your repositories list in the root projects build.gradle:

```gradle
repositories {
    maven { url "https://jitpack.io" }
}
```

Then, add the following to your module dependencies:

```gradle
dependencies {
    compile 'me.oriley:bunyan:0.1'
}
```
